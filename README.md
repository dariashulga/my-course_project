## Применение логистической регрессии для классификации новостных сайтов по типу источника
## 📝 Описание проекта
Программа представляет собой исследовательский инструмент автоматической верификации новостных ресурсов в интересах медиабезопасности. Система позволяет в автоматическом режиме собирать структурно-технические признаки веб-ресурсов и классифицировать их на три целевых класса:
* **Класс 2 (Original):** Авторитетные СМИ с уникальным контентом и открытой редакцией.
* **Класс 1 (Repost):** Ресурсы, занимающиеся частичным или полным копированием с обилием внешних ссылок.
* **Класс 0 (Aggregator):** Автоматические сайты-роботы без оригинального текста.

Интегральная точность (*Accuracy*) разработанной ML-модели на основе логистической регрессии составляет **75.8%**, что на **7.9%** превышает показатели классического жесткого эвристического подхода (67.9%).

## 📁 Структура репозитория
* `1_data_parsing.ipynb` — интерактивный блокнот для сбора признаков с веб-ресурсов.
* `2_model_training.ipynb` — интерактивный блокнот для предобработки, балансировки данных и обучения регрессии.
* `analyzer_site.py` — функциональный модуль (движок парсера), содержащий правила извлечения фич и расчета эвристического рейтинга.
* `config.py` — конфигурационный файл сетевых запросов.
* `requirements.txt` — манифест зависимостей (библиотек).
* `sites_data_labeled1.csv` — верифицированный датасет с экспертной разметкой классов.
* `feature_importance.png` — сохраненный график весовых коэффициентов модели.
